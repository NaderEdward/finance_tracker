Help Page

Log in to a preexisting account and access all the available features. Some features require the user to be logged in.

Register a new account by providing all the necessary information. The email used must be new. The currency the user provides will be used as the currency on which the program is based.

The Balance Page is where the user can add money, remove money or transfer money from a specific balance in their account. The total balance and recent transactions list (showing the latest 5 transactions) are present. At the same time, transferring to another account takes in currency exchange, exchanging the money to their currency before transferring.

The Stats Page uses JavaScript's Chart.js library to visualize statistics on the user's transactions. There is a line graph showing the user's last 5 transactions, an expenses table showing the money spent, added and net value, and a pie chart showing transaction counts based on their reasons.

The Recomenadtions Page requires a minimum of 10 transactions before allowing the user to access it. This is not to provide users with meaningless recommendations. It utilises multiple for loops and if statements to classify the user's transactions (specifically expenses) by their reasons. If a reason makes up 30% of transaction reasons then it's classified as an urgent recommendation, if it makes up 20% of reasons it is classified as an important recommendation.
